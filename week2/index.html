But wait, what is that var thing, and what are those functions??

In Javascript, every variable is declared as var regardless of what type of information it holds. Soon we will discuss datatypes as they relate to JavaScript. JavaScript only works with a few basic variable types: numbers, strings and booleans, along with arrays and objects and functions, which can all be declared with var.

Variables in JavaScript are more easily converted into each other. This is called type coercion and we'll talk about that more next week as well.
